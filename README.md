## Getting Started

1. Clone the repository
2. Follow the setup instructions in the backend documentation
3. Start the application using Docker Compose:
```bash
docker-compose up
```

## Features

- Real-time vehicle tracking
- Interactive subway map
- Delay notifications
- GTFS data integration
- RESTful API
